Краткое описание файлов в этом репозитории.

ledy_maria.py - Проверка кликабельности кнопок и авторизация на Python Selenium.
zara_shop_full - Добавляем несколько товаров в корзину и сравниваем фактическую сумму с суммой в корзине, каталог таваров сохраняем в xls и в json файлы.
test_wb_selenium_def - Проверяем кнопки на сайте Wildberries, далее сохраняем со страницы товары в файл xls.
test_ozon_shop_full - Проверяем кнопки на сайте OZON.
test_wb_selenium_def.py - Проверка активности полей Aviasales.
test_lesson_button_full - Нажатие кнопок и заполнение полей.
test_task - Несколько файлов из практических работ во время обучения.
sushi_nm - Вывод меню с сайта НЕМЕСТНЫЕ, вывод блюд из меню в список ссылок, вывод названия и цены блюд.
letu - тессты кнопок, меню, добавление товаров в корзину и сравнение цен в корзине на сайте Летуаль. С помощью Python Selenium, Playwright, Pytest.
